# 🎙️ EchoRemover
> A Signal Processing project that trains a 1D Convolutional Autoencoder with Spectral Loss to remove echo/reverb from audio files.

---

## 📁 Project Structure

```
Echo_remover/
│
├── cmu_us_bdl_arctic/        ← extracted dataset folder
│   └── wav/                  ← 1150 clean .wav files
│
├── dataset/                  ← generated by echo generator script
│   ├── clean/                ← clean audio files
│   └── echo/                 ← simulated echo versions (same filenames)
│
├── echo_model.pth            ← saved trained model
├── echo_removal.py           ← main training & inference script
└── generate_echo_dataset.py  ← echo dataset generator script
```

## 📥 Dataset

Download the CMU Arctic speech dataset (~150MB):

```
http://www.festvox.org/cmu_arctic/packed/cmu_us_bdl_arctic.tar.bz2
```

### Extract (Windows PowerShell)

```powershell
tar -xjf cmu_us_bdl_arctic.tar.bz2
```

> ⚠️ If `tar` fails on Windows, use **7-Zip** → right-click → Extract Here (do it twice: `.bz2` → `.tar` → folder)

After extracting, wav files will be at:
```
cmu_us_bdl_arctic/wav/
```

---

## 🔧 Generate Echo Dataset

This script takes the clean wav files and simulates room echo using `pyroomacoustics`, creating paired `clean/` and `echo/` folders ready for training.

```powershell
python generate_echo_dataset.py --input_dir cmu_us_bdl_arctic/wav/ --output_dir dataset/ --limit 300
```

**Options:**

| Argument | Default | Description |
|----------|---------|-------------|
| `--input_dir` | required | Folder with clean `.wav` files |
| `--output_dir` | `dataset/` | Where to save `clean/` and `echo/` |
| `--limit` | all files | Max number of files to process |

After running, you'll see:
```
dataset/
    clean/  → 300 files
    echo/   → 300 files (same names)
```

---

## 🏋️ Train

```powershell
python echo_removal.py --mode train --data_dir dataset/ --epochs 50 --save_model echo_model.pth
```

**Options:**

| Argument | Default | Description |
|----------|---------|-------------|
| `--data_dir` | `dataset/` | Path to dataset folder |
| `--epochs` | `50` | Number of training epochs |
| `--batch_size` | `16` | Batch size |
| `--save_model` | `echo_model.pth` | Where to save the model |

Watch the loss go down each epoch — the best model is saved automatically:
```
Epoch [  1/50]  Loss: 0.043000
Epoch [  2/50]  Loss: 0.031000
...
Epoch [ 50/50]  Loss: 0.005200 ✅ best
```

> 💡 If loss is still above `0.01` at epoch 50, increase to `--epochs 100`

---

## 🎧 Remove Echo (Inference)

```powershell
python echo_removal.py --mode infer --input your_audio.wav --output cleaned.wav --load_model echo_model.pth
```

**Options:**

| Argument | Default | Description |
|----------|---------|-------------|
| `--input` | required | Input `.wav` file with echo |
| `--output` | `output_clean.wav` | Output clean `.wav` file |
| `--load_model` | `echo_model.pth` | Path to trained model |

## 🧠 How It Works

| Component | Details |
|-----------|---------|
| **Model** | 1D Conv Autoencoder with skip connections (U-Net style) |
| **Loss** | MSE + Spectral (STFT) loss — prevents near-silence output |
| **Inference** | Overlap-add chunking for smooth reconstruction |
| **Dataset** | Synthetically generated echo using `pyroomacoustics` room simulation |
